Fetch outlet health and report a summary. Call this skill when the user asks about "sources", "outlets", "feeds", "which sources are active", "any failing sources", or similar.

Execute: `bash {baseDir}/scripts/run.sh`

Report: total outlets, active count, inactive count, wire service count, and list any inactive domains by name.
